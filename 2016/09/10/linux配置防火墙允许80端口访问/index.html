linux配置防火墙允许80端口访问

发表于 2016-09-10  |  2016-12-16   |   分类于 Linux

       今天在调试一台centOS系统的服务器时，安装的tomcat配置在80端口，但是通过外网ip访问tomcat服务一直没有响应，而服务器本机上访问localhost是可以的，后来排查发现是服务器的防火墙没有开放80端口的访问，下面就来看一看linux怎么设置防火墙允许访问80端口。

首先，打开防火墙过滤的配置文件：

vim /etc/sysconfig/iptables

然后在22端口的配置语句下添加以下语句：

-A INPUT -m state –state NEW -m tcp -p tcp –dport 80 -j ACCEPT

这样就允许外部请求对于80端口的访问了。
